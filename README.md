# Puuuush

> A modern, minimalist tab manager for Chrome to organize your browser life.

Puuuush replaces your New Tab page with a clean, organized dashboard. Save your open tabs into groups, organize them by project or context, and restore them when you're ready to work. Similar to Toby, but open source and built with a modern stack.

## Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/mtskf/Puuuush.git
   cd Puuuush
   ```

2. **Install Dependencies**
   ```bash
   pnpm install
   ```

3. **Run Dev Server** (for UI development)
   ```bash
   pnpm dev
   ```
   *Note: detailed Chrome API interactions may need mocking or running in the extension environment.*

4. **Build for Production**
   ```bash
   pnpm build
   ```

5. **Load in Chrome**
   - Open `chrome://extensions/`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the `dist` folder generated by the build.

## Project Structure

- `src/features`: Core feature logic (Dashboard, etc.)
- `src/components/ui`: Reusable UI components.
- `src/lib`: Utilities and storage wrappers.
- `dev-docs/`: Developer documentation (Architecture, Specs, Decisions).
